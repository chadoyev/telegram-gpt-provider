## Telegram‑бот с доступом к ChatGPT

Продвинутый Telegram‑бот, который даёт пользователям удобный платный доступ к ChatGPT и другим моделям OpenAI.  
Бот поддерживает **текст**, **голос**, **изображения**, **историю чатов**, **несколько языков интерфейса** и встроенную **платёжную систему**.

### Возможности

- **Доступ к ChatGPT**
  - Поддержка моделей **GPT‑4‑Turbo** и **GPT‑3.5‑Turbo**
  - Учёт лимитов Telegram по длине сообщения
  - Прозрачная тарификация по токенам (вход / выход)

- **Голосовой режим**
  - Приём **голосовых сообщений** от пользователя
  - Расшифровка речи через **Whisper**
  - Ответ бота **голосом** через **TTS‑1**
  - 6 вариантов голоса для каждого языка (RU/KZ/UA/EN)

- **Мультимодальность и картинки**
  - Приём **фотографий** от пользователя и анализ содержимого
  - Поддержка загрузки изображений в **GPT‑4‑Turbo**
  - Отдельный режим **генерации изображений** через **DALL‑E‑3**:
    - пользователь завершает текущий чат и нажимает кнопку генерации
    - размер изображения: `1024x1024`

- **История чатов и отчёты**
  - Хранение истории переписок
  - Возможность получить:
    - **все завершённые чаты**
    - **конкретный чат по номеру**
  - Экспорт чатов и транзакций в **HTML‑файлы**

- **Платежи и биллинг**
  - Баланс пользователя внутри бота
  - Поддержка платёжных провайдеров:
    - **FreeKassa**
    - **Robokassa**
    - **YooKassa**
  - Учёт:
    - токенов (входных и выходных)
    - минут распознавания речи
    - символов озвучки
    - сгенерированных изображений
  - Разные валюты в зависимости от страны:
    - 🇷🇺 Россия — ₽  
    - 🇰🇿 Казахстан — ₸  
    - 🇺🇦 Украина — ₴  
    - 🌍 Другие страны — $

- **Рефералка и бонусы**
  - Триал‑бонус для новых пользователей
  - Бонусы за реферала и реффера
  - Кэшбэк за использование бота

- **Мультиязычный интерфейс**
  - Языки интерфейса:
    - Русский
    - Қазақ тілі
    - Українська
    - English
  - Локализованы:
    - все меню и кнопки
    - тексты ошибок и подсказок
    - приветственные сообщения
    - пользовательское соглашение
    - тарифы и описание цен

- **Удобный интерфейс в Telegram**
  - Главное меню:
    - **👤 Аккаунт**
    - **🟢 Начать чат**
    - **🖼️ Сгенерировать изображение**
  - Меню аккаунта:
    - Пополнение баланса
    - История операций
    - Мои чаты
    - Смена модели ИИ (GPT‑4 / GPT‑3.5)
    - Смена голоса ИИ
    - Смена языка интерфейса
    - Смена страны / валюты
    - Просмотр тарифов
    - Кнопка связи с поддержкой
  - Настройки чата:
    - Включение / отключение озвучивания ответов
    - Завершение активного чата

- **Подписка на канал**
  - Возможность требовать подписку на Telegram‑канал перед использованием бота
  - Встроенные кнопки: «Подписаться» / «Подписался»

- **Админ‑панель**
  - Отдельное меню администратора в Telegram:
    - Статус бота (включён / отключён)
    - Лимит токенов на сообщение
    - Температура (креативность) модели
    - Ручное пополнение баланса пользователей
    - Настройка бонусов, триалов, кэшбэка
    - Отдельные прайсы для GPT‑4, GPT‑3.5, Whisper
    - Переключение на режим бесплатного API
    - Массовая **рассылка** по пользователям
  - Доступ по `ADMIN_ID` и паролю администратора

### Как пользоваться ботом

- **Текстовые сообщения**
  - Перед сообщением нужно ставить **`!`**, чтобы бот понял, что это запрос к ИИ.  
    Примеры:
    - `!Как дела?`
    - `!Составь резюме по этому тексту…`

- **Голосовые сообщения**
  - Отправляете голосовое — бот:
    1. Расшифровывает речь
    2. Генерирует ответ через GPT
    3. При необходимости озвучивает ответ выбранным голосом

- **Фото**
  - Отправляете фото с подписью, начинающейся с `!`
  - Бот анализирует изображение (при включённой модели GPT‑4‑Turbo) и отвечает с учётом содержимого

- **Генерация изображений**
  - Завершаете текущий чат
  - Нажимаете кнопку **«🖼️ Сгенерировать изображение»**
  - Отправляете промпт — бот генерирует картинку через DALL‑E‑3

### Технологии

- **Язык**: Python  
- **Telegram**: `pyTelegramBotAPI` (`telebot`)  
- **OpenAI**: Chat Completions, Whisper, TTS‑1, DALL‑E‑3  
- **Web‑слой**: `Flask` (вебхуки и обработка платежей)  
- **База данных**: PostgreSQL (`psycopg2`)  
- **Платежи**: FreeKassa, Robokassa, YooKassa (`yookassa`)  
- **Прочее**: `tiktoken`, `mutagen`, `schedule`, `requests`

### Быстрый старт

#### 1. Клонировать репозиторий

```bash
git clone https://github.com/your-username/uai_robot.git
cd uai_robot
```

#### 2. Установить зависимости

```bash
pip install -r requirements.txt
```

#### 3. Настроить переменные окружения

Создайте файл `.env` (по аналогии с `.env.example`):

```dotenv
API_TOKEN=your-telegram-bot-token
ADMIN_ID=5400136367
PASSWORD_ADMIN=Admin

OPENAI_API_KEY_PAID=sk-...
OPENAI_API_KEY_FREE=sk-...
OPENAI_API_BASE_PAID=https://api.openai.com/v1/
OPENAI_API_BASE_FREE=https://neuroapi.host/v1/

FREEKASSA_SECRET1=...
FREEKASSA_SECRET2=...
FREEKASSA_MERCHANT_ID=31336
FREEKASSA_IP_SERVER=185.250.205.21

ROBOKASSA_LOGIN=uai_robot
ROBOKASSA_PASS_TEST1=...
ROBOKASSA_PASS_TEST2=...

YOOKASSA_ACCOUNT_ID=209782
YOOKASSA_SECRET_KEY=live_...

DB_HOST=127.0.0.1
DB_USER=postgres
DB_PASSWORD=your-db-password
DB_NAME=db

URL_SUPPORT=https://t.me/uai_support_bot
URL_BOT=https://t.me/uai_robot
URL_CHANNEL=https://t.me/uai_project
```

Убедитесь, что `config.py` считывает эти значения через `os.getenv` (опционально можно использовать `python-dotenv`).

#### 4. Запустить бота

```bash
python main.py
```

Дальше в Telegram настраивается вебхук или long polling — бот начнёт принимать обновления и коллбеки платёжных систем.

### Структура проекта (вкратце)

- **`main.py`**
  - Инициализация Telegram‑бота и Flask‑приложения
  - Обработчики текстов, голосовых, фото
  - Обработка inline‑кнопок и меню
  - Логика биллинга, истории чатов и транзакций
  - Админ‑панель

- **`config.py`**
  - Константы и тексты интерфейса (мультиязычность)
  - Настройки тарифов, сообщений об ошибках, соглашений
  - **Секреты приходят из переменных окружения, а не захардкожены в код**

- **`users/`**
  - HTML‑файлы с историей чатов и транзакций пользователей

### Безопасность

- **Не коммитьте реальные токены и пароли**:
  - Telegram Bot API токен
  - OpenAI API ключи
  - пароли к БД
  - секреты платёжных систем
- Используйте:
  - `.env` (локально)
  - переменные окружения (на сервере / хостинге)
- Если секреты когда‑либо попадали в публичный репозиторий:
  - сразу сгенерируйте новые ключи,
  - при необходимости очистите историю репозитория от старых значений.
